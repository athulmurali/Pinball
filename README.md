# Pinball
A pinball game that has a ball and a racket built with  Racket (a functional programming language) 

To execute this, 
install Racket
build the racekt file

------------------------------------------------
Game instructions:
===============================================

To add a ball into the game, press b
move up -      up 

move down -    down

move left -    left

move right -   right

pause -        space

===============================================
